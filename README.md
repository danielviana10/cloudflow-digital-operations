# CloudFlow Technologies

## Sobre a empresa

A CloudFlow Technologies é uma empresa SaaS que oferece uma plataforma de gestão empresarial baseada em nuvem.

Os clientes acessam a plataforma através de:

- CloudFlow Web
- CloudFlow Mobile

A comunicação é realizada através da CloudFlow API.

---

## Produtos

### CloudFlow Web

Portal principal utilizado pelos clientes.

### CloudFlow Mobile

Aplicativo mobile para Android e iOS.

### CloudFlow API

API responsável pela integração entre sistemas externos e a plataforma.
